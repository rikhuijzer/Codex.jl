add_basic_motives!(df::DataFrame)

Add basale drijfveren columns and remove the items. All items are 1 (helemaal niet) to 7 (in extreem sterke mate).

1-4 autonomie;
5-8 verbondenheid;
9-12 competentie;
13-16 structuur;

Amotivation (3,11,16);

Introjected regulation (7,12,18);
Identified regulation (5,9,15);

Intrinsic motivation (2,10,14)

The items start at 48 for all versions.

Also, 2019 to 2020 go from 1 to 7 so let's say all.

Codex.Questionnaires.add_temperaments! — Method
add_temperaments!(df)

Add approach and avoidance temperament columns and remove the items.

Scoring is based on Ruud his Word document:

Approach temperament = item2+item4+item5+item8+item10+item11
Avoidance temperament = item1+item3+item6+item7+item9+item12

And according to 2019-first/questionnaires/hotel.csv the items start at 62. Also for 2020 first.

Codex.Questionnaires.join_questionnaires — Method
join_questionnaires(raw_dir::String, questionnaires::Array{String,1}, groups::Array{String,1})::DataFrame

Joines multiple questionnaires for the members of groups, where groups ensures that the joins do not remove rows which shouldn't be removed. For instance, given questionnaires "A" and "B" containing respectively columns :a1, :a2 and :b1, this method will return { group, id, A_a1, A_a2, B_b1 }.

Codex.Questionnaires.self_efficacy2scores — Method
self_efficacy2scores(echo::DataFrame)::DataFrame

Return the scores for the echo questionnaire, which is about self-efficacy. None of the items appear to be reversed, so this method just returns the sum.

This function assumes questions [1, 14] which holds for all questionnaires in from 2018 to the time of writing (2021).
